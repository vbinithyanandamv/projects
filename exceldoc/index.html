VizExcel

VizPDF is a Excel generating plugin library which lets you export excel from the config provided.

Features
  • Conditional Formatting
  • Style Formatting
  • Native Font Family Support
  • Number Formatting
  • Browser and Node support

Installation

Install vizExcel with npm

  npm install @visualbi/viz-excel

Usage/Examples

Browser

 
import VizcsvBuilder from '@visualbi/vizexcel';
import { data } from './data'; //your data config file

async function main() {
    try {
        const vizcsv = new VizcsvBuilder(data);
        await vizcsv.generate();
        await vizcsv.save('generated');
        return;
    } catch (error) {
        console.error(error);
    }
}

Run Locally

Clone the project

  git clone https://link-to-project

Go to the project directory

  cd vizexcel

Install dependencies

  npm install

For browser

  npm run sample-basic

For Node

  npm run sample-node
